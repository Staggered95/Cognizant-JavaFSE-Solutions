# Cognizant Digital Nurture 5.0 - Java FSE Solutions

A complete repository containing source code, configurations, and execution proofs for the Cognizant Digital Nurture 5.0 Deep Skilling mandatory hands-on assignments.

## 📁 Curriculum Overview

The coursework is structured chronologically by week, covering the full Java Full-Stack Engineering (FSE) pipeline:

| Module | Technologies Covered |
| :--- | :--- |
| **Week 1** | Data Structures & Algorithms, Design Patterns & Principles |
| **Week 2** | PL/SQL Programming, SL4J Logging, TDD (JUnit & Mockito) |
| **Week 3** | Spring Core & Maven, Spring Data JPA |
| **Week 4** | Spring REST & Spring Boot |
| **Week 5** | Microservices Architecture |
| **Week 6** | React Single Page Applications (SPA) |
| **Week 7** | Git Version Control, Docker, Cloud Fundamentals |

## 📸 Documentation & Naming Conventions

To maintain an organized tracking system for assignment evaluations, execution proofs and screenshots follow a strict suffix convention:

* `_A`, `_B`, `_C`, etc. &rarr; **Process & Implementation:** Sequential screenshots demonstrating the workflow, IDE configuration, or code structure for a specific module.
* `_O` &rarr; **Execution Output:** The final compiled result, UI render, or successful terminal output.
* `_O1`, `_O2`, etc. &rarr; **Multiple Outputs:** Used when an assignment requires demonstrating multiple states, conditional renders, or distinct terminal results for the same task.
